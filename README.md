# Senda.

Senda es una plataforma diseñada para convertir conversaciones en transacciones comerciales validadas. Automatiza el ciclo de vida de la facturación electrónica integrándose directamente en WhatsApp, permitiendo que la inteligencia artificial gestione la operación fiscal con mínima intervención humana.

## 🚀 Tecnologías utilizadas
Este proyecto fue construido íntegramente mediante ingeniería de prompts avanzada, utilizando **Gemini** como arquitecto y copiloto de programación.

* **Backend**: Node.js, TypeScript.
* **Base de datos**: Supabase, PostgreSQL, Prisma ORM.
* **IA**: Google Gemini API, Vertex AI.
* **Mensajería**: Twilio (WhatsApp API).
* **Almacenamiento**: Supabase Storage.

## 💡 Cómo funciona
Senda actúa como un agente inteligente que reside en WhatsApp. Los usuarios pueden subir sus certificados digitales (.cer) de forma segura y, a través de una interfaz conversacional natural, la IA procesa la información para emitir facturas legales.

## 🛠 Instalación y Configuración
1. Clona el repositorio: `git clone https://github.com/[TU_USUARIO]/Senda.git`
2. Instala dependencias: `npm install`
3. Configura tus variables de entorno en un archivo `.env` (basado en `.env.example`).
4. Ejecuta el servidor: `npm run dev`
